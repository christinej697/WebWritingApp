DLR 4: Dispelling the Constant Voltage Drop Error

Let's start off with a question. Select all of the following that are true as the value of R2 decreases in the circuit below.

Both of the above statements are false, and notice, they are based on the assumption that the voltage drop across R2 or R3 remains constant as the value of R2 decreases. The only element in the circuit that has a potential difference that does not change as the value of R2 changes is the ideal independent voltage source. Indeed, this is in essence the definition of such a source -- it provides a constant potential difference across its terminals regardless of the circuit to which it is connected. We cannot extend the idea of a constant potential difference to the remainder of the elements however, as we will see through the following argument. Let's redraw the circuit, labeling potential differences.

Point 1: Let's start by assuming (incorrectly) that the voltage drop across resistor R3 remains constant. If this is the case, since the value of R3 does not change in the stated problem, Ohm's law suggests that the current in resistor 3 is constant. That is, I3 = V3/R3 would be a constant. Again, this is an incorrect statement for this circuit!

Point 2: Since R2 and R3 are tied together at both nodes, they are in parallel. Elements in parallel, share a common potential difference and so V2 = V3. These two sentences are correct.

Point 3: Since we argued in Point 1 that the potential across resistor 3 remains constant, and as Point 2 indicates, resistor 2 and resistor 3 have the same potential difference, V2 = V3 = a constant. This would suggest that as the value of R2 decreases, Ohm's law reveals that the current in R2 increases as I2 = V2/R2.

Point 4: Applying KCL to the node connecting the three resistors, we see that I1 = I2 + I3. This is a correct application of KCL. Since we have I3 as a constant and I2 as increasing, I1 must increase to satisfy KCL. Ohm's law applied to resistor R1 requires V1 = I1R1. Since R1 remains constant and I1 increases, this would suggest that V1 would have had to increase to drive a larger current through R1. This seems to go against the original notion of the voltage drop across a resistor not changing as R2 changes, but let's run with the idea and see where it takes us.

Point 5: So our arguments have led us to the following: V1 increasing and V2 and V3 remaining constant as the value of R2 decreases.
